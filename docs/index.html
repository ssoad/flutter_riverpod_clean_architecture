Flutter Riverpod Clean Architecture

Production-ready Flutter template with advanced features and clean architecture

Getting Started

This documentation covers all aspects of the Flutter Riverpod Clean Architecture template, a production-ready solution for building scalable, maintainable Flutter applications. 

git clone https://github.com/ssoad/flutter_riverpod_clean_architecture.git
cd flutter_riverpod_clean_architecture
flutter pub get

You can use this project as a template:

  1. Go to the GitHub repository
  2. Click on "Use this template" button
  3. Follow the GitHub instructions to create your repository
  4. Clone your new repository locally
